# Fx chart matching

## 仮想環境有効化
```
 . /venv/bin/activate
```
## 仮想環境無効化
```
deactivate
```
## 仮想環境ライブラリ一覧書き出し
```
pip freeze > requirements.txt
```
## 仮想環境ライブラリ一覧インストール
```
pip install -r requirements.txt
```
## メインアプリケーション実行
```
python -m fx-chart-matching
```
## テスト実行
```
python -m unittest discover tests
```
## git ステージする
```
git add --all
```
## git コミット
```
git commit -m 'first commit'
```

```
git remote add origin https://github.com/shimadahanz0/fx-chart-matching.git
git push origin master
```
